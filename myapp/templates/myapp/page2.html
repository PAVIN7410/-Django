Django и Flask используют один шаблонизатор — Jinja. При работе мы будем перемещаться среди html-страниц. Для работы с Jinja нужен файл, который будет являться основой, заготовкой, шаблоном. Есть два распространённых варианта его названия: base.html и layoute.html. В этот файл мы вынесем повторяющиеся части файлов. В папке main, содержащей html-файлы, создаём файл layoute.html. Создаём блоки, которые будут заполняться:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta http-equiv="X-UA-Compatible" content="ie=edge"> % block title % # все блоки в фигурных скобках{} %endblock % </head> <body> % block content % % endblock % </body> </html>
